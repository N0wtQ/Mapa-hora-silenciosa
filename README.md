# 🤫 Mapa de Espacios Silenciosos · España

Mapa interactivo de lugares tranquilos y establecimientos con **hora silenciosa** en España. Dirigido a personas con TEA, hipersensibilidad sensorial, ansiedad u otras condiciones que se benefician de entornos de baja estimulación.

---

## Estructura del proyecto

```
mapa-silencioso/
├── index.html                    ← Mapa interactivo completo (abre en el navegador)
```

---

## Uso rápido

### Ver el mapa
Abre `index.html` directamente en el navegador. No requiere servidor.

---

## Colaborar

Para sugerir un nuevo lugar, es imprescindible incluir una URL de fuente pública (web oficial, noticia de prensa, memoria de RSC). Los lugares sin fuente verificable se marcarán como "PENDIENTE".
